# Module 1: ROS 2 Fundamentals

Welcome to the ROS 2 Fundamentals module of the Physical-AI & Humanoid-Robotics textbook. This module introduces you to the Robot Operating System (ROS 2), which is a flexible framework for writing robot software. 

## Learning Objectives

By the end of this module, you will be able to:

- Understand the core concepts of ROS 2 architecture
- Create and run nodes, topics, services, and actions
- Integrate Python with ROS 2 using rclpy
- Describe robots using Unified Robot Description Format (URDF)
- Connect AI systems with robotic platforms
